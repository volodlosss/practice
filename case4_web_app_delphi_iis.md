# Кейс 4: Веб-приложение (Delphi 10.2 + IIS + MS SQL Server)

## Анализ
- На рынке популярны CRM/ERP, интернет-магазины, корпоративные порталы.
- Все они используют клиент-серверную архитектуру, масштабируемость и API-интеграции.

## Учебное приложение
Тема: система бронирования туров.

### API
- GET /api/tours — список туров
- POST /api/orders — оформить заказ
- GET /api/orders/{id} — детали заказа

### Настройка IIS
1. Установить IIS и Delphi приложение.
2. Настроить пул приложений и подключение к БД.
3. Включить HTTPS.
