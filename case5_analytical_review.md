# Кейс 5: Аналитический обзор (БД Туризм)

- **Функциональность**: хранение данных о клиентах, турах, заказах.
- **Производительность**: индексы по ключам.
- **Юзабилити**: простая структура.
- **Безопасность**: внешние ключи и ограничения.
- **Масштабируемость**: легко добавить новые таблицы (отели, билеты).
- **Сопровождаемость**: структура и документация понятна.
- **Переносимость**: SQL адаптируется под другие СУБД.
- **Качество кода**: читаемость, стандарты именования.
- **Тестирование**: проверки insert/select.
